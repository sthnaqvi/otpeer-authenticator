# Tray

Implemented in `packages/desktop` — compact last-used popup (not the old multi-code menu).

**Clipboard:** OTP copy saves prior clipboard and restores it after ~45s if the clipboard still holds that OTP (`electron/clipboard_otp.ts`). Used by tray and in-app OTP copy.
